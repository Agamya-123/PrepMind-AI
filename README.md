# PrepMind AI

PrepMind AI is a lightweight exam-preparation web app built as a single HTML page.

It provides:

- exam overview generation
- topic-wise teaching
- practice question generation
- doubt solving chat
- simple progress tracking

## Live App

If GitHub Pages is enabled for this repository, the app will be available at:

`https://agamya-123.github.io/PrepMind-AI/`

## Project Structure

- `index.html` - entry point for deployment
- `PrepMind_AI (2).html` - main application file

## Features

- Gemini and OpenAI API key support
- automatic Gemini model fallback on quota issues
- structured JSON question generation
- client-side progress tracking
- no build step required

## Run Locally

1. Clone the repository.
2. Open `index.html` or `PrepMind_AI (2).html` in a browser.
3. Enter your API key.
4. Start using the app.

## Deployment

This project can be deployed as a static site on:

- GitHub Pages
- Netlify
- Vercel

For GitHub Pages:

1. Go to repository `Settings`.
2. Open `Pages`.
3. Set source to `Deploy from a branch`.
4. Select branch `main`.
5. Select folder `/ (root)`.
