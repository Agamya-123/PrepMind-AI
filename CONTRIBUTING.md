# Contributing to PrepMind AI

Thanks for contributing.

## How To Contribute

1. Fork the repository.
2. Create a feature branch.
3. Make focused changes.
4. Test the app in a browser.
5. Open a pull request with a clear summary.

## Contribution Guidelines

- Keep the app static and easy to deploy.
- Prefer small, reviewable changes.
- Preserve the current UI style unless the change is intentional.
- Avoid breaking Gemini and OpenAI support.
- Test practice generation, overview, and chat flows before submitting.

## Pull Request Checklist

- The app loads without console-breaking errors.
- Main user flows still work.
- Documentation is updated if behavior changed.
- Changes are scoped to the problem being solved.

## Reporting Issues

When opening an issue, include:

- what you expected
- what happened instead
- browser used
- screenshots if relevant
- console errors if available
